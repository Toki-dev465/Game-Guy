# Game-Guy
A compact handheld console featuring a 0.96 inch OLED display, a passive piezo buzzer for sound effects, D-pad, and A and B buttons. It is powered by an LiPo battery and the XIAO RP2350 microcontroller. 

# BOM
1x XIAO RP2350
6x TS02 tactile switch
1x I2C OLED Display Module 0.96 Inch SSD1306
1x Piezo buzzer
1x Lithium-Ion Polymer (LiPo) Battery (3.7V 110mAh)
1x Slide switch
