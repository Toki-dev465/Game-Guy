# Game-Guy
A compact handheld console featuring a 0.96 inch OLED display, a passive piezo buzzer for sound effects, D-pad, and A and B buttons. It is powered by an LiPo battery and the XIAO RP2350 microcontroller. Right now, the only game is flappy bird. Simply tap A, B or D-pad Up to flap the bird.
